# Fortnite Shop Telegram Bot

Бот для автоматического отслеживания магазина Fortnite.

## Возможности
- Автоматическая отправка магазина в 3:00 МСК
- Просмотр текущего магазина по команде /shop
- Отдельный просмотр ежедневных предметов и новинок
- Интерактивные кнопки

## Команды
- /start - Начало работы
- /shop - Текущий магазин
- /daily - Ежедневные предметы
- /featured - Новинки
- /help - Помощь
